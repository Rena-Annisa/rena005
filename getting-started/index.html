    No 1¶

    Buat program deklarasi 5 variabel dan lakukan perhitungan rata-rata nilai tsb. Tampilkan hasilnya!

    Jawaban:

    data1 = int(input("Masukkan data ke-1 = "))
data2 = int(input("Masukkan data ke-2 = "))
data3 = int(input("Masukkan data ke-3 = "))
data4 = int(input("Masukkan data ke-4 = "))
data5 = int(input("Masukkan data ke-5 = "))
n = 5
rata_rata = (data1 + data2 + data3 + data4 + data5)/n
print ("Hasil rata-rata : ", rata_rata)

    Output:

    Masukkan data ke-1 = 5

Masukkan data ke-2 = 7

Masukkan data ke-3 = 8

Masukkan data ke-4 = 9

Masukkan data ke-5 = 12
Hasil rata-rata :  8.2

    No 3¶

    Buat program untuk menghitung Index Massa Tubuh/BMI (Body Massa Index). Hitung BMI, kemudian tentukan kelompok berat badan.

    Di bawah 18.5 = berat badan kurang

    18.5 - 22.9 = berat badan normal

    23 - 29.9 = berat badan berlebih (kecenderungan obesitas)

    30 ke atas = obesitas

    Anda termasuk yang mana?

    Jawaban:

    print ("---Menghitung Index Massa Tubuh---")
berat = float (input("Masukkan berat badan (kg) : "))
tinggi = float (input("Masukkan tinggi badan (cm) : "))
rumus = berat/((tinggi/100)**2)

if rumus < 18.5:
    print ("Index Massa Tubuh : ", rumus)
    print ("Berat Badan Anda Kurang!")
elif rumus <= 22.9:
    print ("Index Massa Tubuh : ", rumus)
    print ("Berat Badan Anda Normal.")
elif rumus <=29.9:
    print ("Index Massa Tubuh : ", rumus)
    print ("Berat Badan Anda Berlebih (Kecenderungan Obesitas).")
else:
    print ("Index Massa Tubuh : ", rumus)
    print ("Anda Obesitas!")

    Output:

    ---Menghitung Index Massa Tubuh---

Masukkan berat badan (kg) : 50

Masukkan tinggi badan (cm) : 153
Index Massa Tubuh :  21.35930624973301
Berat Badan Anda Normal.

    Berdasarkan output di atas, berat badan Saya termasuk normal.

    No 5¶

    Buat program untuk menentukan validasi username dan password, dimana akan diulang maksimal 3 kali. Jika benar akan muncul komentar "anda berhasil masuk" dan jika tidak akan muncul komentar "maaf usename dan password anda salah".

    Jawaban:

    username = "rena"
password = 18012000
n = 3
while n > 0:
    val_name = str(input("Masukkan Username : "))
    val_pass = int(input("Masukkan password : "))
    if val_name == username and val_pass == password:
        print ("Selamat Anda Berhasil Masuk!")
        break
    else:
        n = n-1
        print ("Maaf Username dan Password yang Anda Masukkan Salah.")

    Masukkan Username : rena

Masukkan password : 18012000
Selamat Anda Berhasil Masuk!
